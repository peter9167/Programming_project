[ ] 연산자의 역할

연산의 우선순위 : http://www.tcpschool.com/codingmath/priority

-----------------------------------
int arr[5] = {1, 2, 3, 4, 5};

printf("a[3] : %d \n", arr[3]);		결과 : 4
printf("*(a+3) : %d \n", *(arr + 3));	결과 : 4
-----------------------------------

결론부터 말하면 arr[3] 과 *(arr + 3) 은 같다.

arr[3] 이라는 것은 arr 배열 중 4번째 값을 가르키는 것
*(arr + 3) 은 arr 메모리(arr[0])에서 3을 더하여 arr[3] 값을 출력
