C언어 버퍼란

- 특정 크기의 메모리 공간을 의미
- 운영체제는 입력과 출력을 돕는 '입출력 버퍼' 를 생성하여 제공합니다.
- 표준 입출력 함수를 기반으로, 데이터 입출력 시 입출력 버퍼를 거칩니다.


Stdin 이란?

- 수 많은 버퍼 중에서도 키보드의 입력을 처리하는 버퍼는 바로 입력 버퍼, 혹은 stdin 이다. 
- 키보드로 쳐다는 모든 정보는 일시적으로 stdin 에 저장되었다가 나중에 입력이 종료되면 한꺼번에 처리하는 것


scanf 함수에 동작 원리

- scanf 함수는 ' ', '\n', '\t' 입력 될 때까지 stdin 으로부터 데이터를 얻는다 (입력 종류)



문자열 리터럴(literal)에 대한 이해

- 프로그래밍 언어에서 리터럴(literal)이란, 소스 코드 상에서 고정된 값을 가지는 것을 일컫음.
- C 언어의 경우 큰 따옴표(") 로 묶인 것들을 문자열 리터럴(string literal) 이라 불림.

------------------------
char str[] = "sentence"; // sentence 라는 문자열을 str 이라는 배열 대입
char *pstr = "sentence"; // 아래 자세하게 설명
------------------------

- "sentence" 는 문자열이 저장된 주소값을 말하며 char *pstr = "sentence"; 를 실행하면
- "sentence의 시작 주소값 좀 가져와서 pstr 에 대입해라" 라는 의미의 작업 실행
- 따라서, pstr 은 "sentence" 라는 리터럴을 가리키고 printf("%s", pstr) 을 했을 때 sentence를 출력

- 리터럴은 소스 코드 상에서 고정된 값을 가지는 것들이기 때문에 전에는 위와 같은 코드로 선언했었지만
- 최근에는 const char* 로 가르켜서 선언하는 특징을 가진다. 

****** pstr[1] = 'a'; 리터럴은 오직 읽기만 가능하기 때문에 본 코드처럼 수정하거나 쓰려면 오류가 발생한다. 
****** printf("pstr : %s \n", pstr); 출력은 읽기만 하기 때문에 정상적으로 코드가 실행된다. 



문자열 처리 함수 정리

1. strlen() 함수	: 인수로 전달된 문자열의 길이를 반환하는 함수

2. strcat(), strncat() 함수 : strcat()함수와 strncat() 함수는 하나의 문자열에 다른 문자열을 연결해주는 함수

3. strcpy(), strncpy() 함수 : strcpy() 함수와 strncpy() 함수는 문자열을 복사하는 함수

4. strcmp(), strncmp() 함수 : strcmp() 함수와 strncmp() 함수는 문자열의 내용을 비교하는 함수

5. atoi(), atol(), atoll(), atof() 함수 : 인수로 전달된 문자열을 해당 타입의 숫자로 변환시켜주는 함수

6. toupper(), tolower() 함수 : 인수로 전달된 문자열의 영문자를 모두 대문자나 소문자로 변환시켜주는 함수
